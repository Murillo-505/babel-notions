# notions-of-babel
Uma aplicação de organização de conhecimento baseada na metáfora de bibliotecas físicas, inspirada na estrutura da *Library of Babel*.<br/>
Diferente de sistemas de notas tradicionais em pastas, o notions-of-babel organiza o conhecimento através de uma hierarquia física intuitiva:
Usuário -> Parede -> Estante -> Prateleira -> Volume (Notas).
O objetivo é permitir que o usuário "perca-se" navegando por suas próprias ideias ou encontre uma agulha no palheiro através de uma busca profunda e instantânea.<br/>

1. **Parede:** O nível mais alto (ex: Trabalho, Pessoal, Estudos Acadêmicos).
2. **Estante:** Tópicos amplos (ex: Computação, Filosofia, Culinária).
3. **Prateleira:** Subtópicos específicos (ex: Banco de Dados, Estoicismo, Massas).
4. **Volume:** O "livro" de anotações propriamente dito (ex: MongoDB, Sêneca, Receitas de Verão).

O sistema opera em dois modos distintos de recuperação de dados:<br/>

* Busca Livre (Navegação): Interface visual para navegação manual entre estantes e prateleiras. Ideal para revisões casuais e exploração de temas relacionados.
* Busca Profunda (Citação): Inspirado na *Library of Babel*, o sistema indexa o conteúdo textual de cada volume. O usuário pode buscar por uma frase exata ou termo, e o sistema retornará o volume exato onde aquela informação reside.


O sistema de anotações funciona como um bloco de notas clássico, podendo ser alterado de acordo com a necessidade do usuário.
O objetivo é ter uma extrutura dinâmica e que funcione para organizar as infinitas possibilidades de anotações, porém mantendo um caminho sólido para buscas detalhadas, sem ser necessário checar arquivo por arquivo para verificar o conteúdo requerido.
